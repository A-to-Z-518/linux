linux命令
	ctrl+alt+t：打开终端
	ls：显示文件
	cp+文件+文件夹：复制文件到文件夹中
	cd：双击
	cd -：返回上一级
	cd ~：返回用户主目录
	cd /home/linux：绝对路径
	cd ./linux：相对路径
	touch+名称：创建文件
	mkdir+名称：创建文件夹
	rm+名称：删除文件
	rm+-r+名称：删除文件夹  
	tab：补齐文件名或者命令
	ctrl+l：清屏
	pwd：列出当前所在路径
	ls+-a：列出当前目录下的所有文件，包括隐藏文件
	r：可读    w：可写     x：可执行
	mv+文件名+文件夹：将文件向下一级移动
	mv+文件名+../：将文件移到上一级移动
	mv+文件名+新文件名：将文件重命名成新文件名

vim+文件名：命令模式   
        按ESC输入命令
	yy：复制一行
	yn：复制n行，将光标移动到要复制的第一行，n=要复制的行数
	v-y：选择复制，光标在复制内容开始按v，在复制内容结尾按y
	p：粘贴到光标当前位置的下一行，选择复制时在光标位置复制
	u：撤销
	dd：剪切一行
	dn：剪切n行
	nG：跳转，n表示要跳转到的行数
	:w  ：保存
	:q   ：退出
	:q！：强制退出（不保存）
	:wq  ：保存并退出
	:wq！：强制保存退出
	/内容：实现查找功能

gcc编译器：gcc是一个跨平台跨软件的编译器

         过程：

	1、gcc+源文件名+-o+生成文件名

	2、./生成文件名.sh           // 用"./生成文件名"显示输出内容

shell脚本：用来监视电脑状态，用作批量处理。linux中指定的解释器是bash

          过程：

	1、vim  文件名.sh ：创建脚本文件，以.sh结尾

	2、date ：输出当前日期
	     echo" "：输出字符串
	     cd+文件夹：进入文件夹
	     touch+文件名：创建文件记录事件

	3、chmod  +x  文件名.sh：给脚本文件增加执行权限
	
	4、./文件名.sh：输出显示脚本文件

shell脚本定义变量：变量名尽量大写
	变量名=mu ：shell脚本中没有变量名
	echo $变量名 ：$+变量名是对变量进行赋值操作
	echo "name ， $变量名" ：" "不完全输出，" "会先将命令解析后输出
	echo 'name ， $变量名'：' '完全输出，' '中的内容原样输出

输入： read -p "显示文字：" 变量名     //-p表示先输出后输入
           echo "name is $变量名"

脚本逻辑表达式：通常与 if 组合使用

	1、判断一个整数是不是大于另一个整数
	      -eq 判断两个数是不是相等   equal 
	      -gt 判断一个数是不是大于另一个数
	      -lt 判断一个数是不是小于另一个数

	2、判断文件存不存在
	      -e $FILE    # -e 判断文件是否存在  exist
	      -d 判断文件是不是一个目录	      	
 	      -f 判断此文件是不是普通文件(文本文件， 可执行文件)  
	      -x 判断此文件是否有执行权限	 
	      -r 判断此文件是否有读权限		
 	      -w 判断此文件是否有写权限
